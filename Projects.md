# Library App
My third console application considered to the library and books. In this project I use `class` and principles of OOP. 

As previous project this one has same command options:
- [+] - Add book
- [-] - Remove book
- [A] - Show all books
- [E] - Edit book
- [S] - Search book
- [Q] - Quit app
